**TorrFLIX** — это высокопроизводительный асинхронный агрегатор торрент-раздач. Приложение объединяет мощь **Jackett** для поиска по сотням трекеров и эстетику **TMDB** для визуализации вашей медиатеки. 

Построен на базе **FastAPI**, что обеспечивает мгновенный отклик и поддержку современных веб-стандартов.

---

## 🔥 Ключевые особенности

* **🚀 Полная асинхронность:** Бэкенд на FastAPI (Python 3.10+) обеспечивает параллельную обработку запросов к API и парсинг раздач.
* **🎭 Интерфейс уровня Premium:** Чистый, адаптивный дизайн с поддержкой D-Pad (управление стрелками клавиатуры или пультом TV).
* **🖼 Image Proxy & Cache:** Автоматическое кэширование постеров на сервере для обхода блокировок и мгновенной загрузки.
* **📺 Трейлеры:** Встроенный просмотр официальных трейлеров с YouTube прямо в карточке фильма.
* **🔐 Защищенная админка:** Управление API-ключами через веб-интерфейс с Basic Auth авторизацией.
* **🧲 Умные магниты:** Автоматическое извлечение Magnet-ссылок из `.torrent` файлов на лету.
* **🌓 Dark/Light темы:** Поддержка темного и светлого режимов с сохранением выбора.

---

## 🛠 Технологии

* **Backend:** FastAPI, Uvicorn, Httpx, Aiofiles, Pydantic.
* **Frontend:** Vanilla JS (ES6+), CSS3 (Modern Glassmorphism), HTML5.
* **Database:** JSON-based config (Zero-DB approach).
* **DevOps:** Docker, Docker Compose.

## 🚀 Быстрый старт

### 1. Установка
```bash
git clone https://github.com/ByteBudda/torrFlix.git
cd torrFlix
```

2. Запуск через Docker Compose
Приложение автоматически поднимется на порту 8800.
```
docker compose up -d --build
```
4. Первичная настройка
 * Откройте админ-панель: http://ваш-ip:8800/admin
 * Введите логин/пароль (по умолчанию настроены в коде/окружении).
 * Укажите ваши ключи TMDB API и Jackett URL/Key.
 * Сохраните и наслаждайтесь!
📂 Структура проекта
 * server.py — Асинхронный сервер FastAPI.
 * index.html — Одностраничное приложение (SPA) с логикой навигации.
 * cache_img/ — Хранилище закэшированных постеров (локально).
 * config.json — Конфигурация приложения (создается автоматически).
 * docker-compose.yml — Инструкции для развертывания в контейнере.
🛡 Безопасность
Все чувствительные данные (ключи, пароли, кэш) добавлены в .gitignore. Приложение готово к деплою на публичные серверы без риска утечки учетных данных.
